Event Sourcing library in Go

## Overview

`pupsourcing` is a Go library for implementing event sourcing patterns in your applications. Event sourcing is a pattern where state changes are stored as a sequence of events, providing a complete audit trail and enabling powerful features like time travel, event replay, and complex event processing.
